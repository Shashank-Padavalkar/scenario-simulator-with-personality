# Scenario Simulator – AI Decision Intelligence

Scenario Simulator is a full-stack web application that helps users evaluate decisions by simulating possible outcomes using AI.

The application uses a Retrieval-Augmented Generation (RAG) pipeline to analyze a user’s input decision and generate structured insights, including:

- Best-case outcome  
- Worst-case outcome  
- Most likely outcome  
- Potential risks  
- Actionable recommendations  

It combines a FastAPI backend with the Google Gemini API and a local vector database (ChromaDB) to provide context-aware, intelligent responses.

The goal of this project is to support smarter decision-making through structured AI-driven scenario analysis.
